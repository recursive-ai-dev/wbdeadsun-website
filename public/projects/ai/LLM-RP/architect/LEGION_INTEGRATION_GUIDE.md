## Berserker × Architect: The Debug-Synthesis Loop

### Scenario: Production Incident

```
PHASE 1: BERSERKER (Execution)
├── Exception Harvest on error
├── Temporal Backtrack to find origin
└── Logic Shard Sequence to isolate cause
    └── Output: Root cause identified, patch ready

PHASE 2: ARCHITECT (Synthesis)
├── Step-Back: Why did this class of bug occur?
├── Skeleton: What systemic factors enabled it?
├── ToT: How might it recur? How to prevent?
├── Evaluation: Which prevention strategy is best?
├── Synthesis: Institutional learning document
└── Calibration: Confidence in prevention

PHASE 3: BARD (Chronicle)
├── Document the incident
├── Record the ritual law
└── Update institutional memory

OUTPUT: Bug fixed + systemic prevention + knowledge preserved
```

### Integration Points

| Berserker Output | Architect Input | Architect Output | Berserker Input |
|-----------------|-----------------|------------------|-----------------|
| Shard plan | Design verification protocol | Verification checkpoints | Follow checkpoints |
| Root cause | Abstract to principle | Prevention strategy | Implement prevention |
| Patch | Evaluate for side effects | Risk assessment | Refine patch |
| Postmortem | Synthesize lessons | Institutional learning | Apply to next bug |

### Cross-Server Invocation Pattern

```typescript
// Example: Debug with full Legion support

// 1. Architect designs the approach
const approach = await architect.architect_open_synthesis({
  dataset_description: "Production error in payment service",
  domains: ["technical", "economic"],
  stakes: "Payment processing downtime"
});

// 2. Berserker executes the debug
const debugResult = await berserker.berserker_logic_shard_sequence({
  error: errorMessage,
  stack_trace: stackTrace
});

// 3. Hound verifies the fix
const verification = await hound.hound_verify_fix({
  original_error: errorMessage,
  patch: debugResult.patch,
  test_cases: testCases
});

// 4. Bard documents the ritual
await bard.bard_chronicle_fix({
  error: errorMessage,
  root_cause: debugResult.rootCause,
  patch: debugResult.patch,
  ritual_law: debugResult.ritualLaw
});

// 5. Architect synthesizes learnings
const learnings = await architect.architect_synthesize({
  session_id: approach.session_id
});
```
